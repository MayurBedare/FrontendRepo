# AI Knowledge Hub - Frontend

A premium Knowledge Sharing Platform built with React, featuring a sleek design and integrated AI tools to help authors create high-quality content.

## ‚ú® Approach & Key Design Decisions

### Design System
- **Glassmorphism:** Adopted a modern, premium aesthetic with subtle blurs and translucent layers.
- **Vanilla CSS:** Custom styles were used to ensure maximum control over the visual identity without the overhead of heavy CSS frameworks.
- **Lucide Icons:** Used for a clean and consistent iconography throughout the app.

### Key Features
- **Context API:** Managed global authentication state efficiently.
- **React-Quill:** Integrated a rich text editor for a seamless authoring experience.
- **AI Sidebar/Tools:** Integrated specific UI actions to trigger AI content improvements and tag suggestions.
